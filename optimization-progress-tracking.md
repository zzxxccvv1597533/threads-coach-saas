#### 2.2.3 Avoid-list（禁止句式清單）
- [x] 開頭句式（「你是不是也這樣？」「你有沒有發現...」等）
- [x] 過渡句式（「其實...」「說真的...」「不得不說...」等）
- [x] 結尾句式（「你覺得呢？」「留言告訴我」等）
- [x] AI 特徵詞（「作為一個...」「首先...其次...最後...」等）
- [x] 填充詞（「事實上」「基本上」等）
- [x] 共 26 個預設禁止句式

#### 2.2.4 AI Detector（AI 痕跡檢測）
- [x] Avoid-list 匹配（35% 權重）- 檢查是否包含禁止句式
- [x] 重複模式檢測（25% 權重）- 檢查是否有重複的句式結構
- [x] 典型 AI 短語（25% 權重）- 檢查是否有常見的 AI 用語
- [x] 句式密度（15% 權重）- 檢查特定句式的出現頻率
- [x] 分數等級和動作建議

#### 2.2.5 Observability（可觀測性）

- [x] logError() - 記錄錯誤
