```

```
5 個主題建議
    ↓ 轉成 Embedding
    ↓ 計算兩兩相似度
    ↓ 如果 > 80% 相似，替換掉重複的
```

**效果**：
- 確保 5 個主題都是不同角度
- 減少「看起來不同但意思相近」的情況

    ↓ 顯示「相似草稿群組」
```

**效果**：
