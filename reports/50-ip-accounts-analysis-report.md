## 二、領域識別技術說明

### 技術實作方式：關鍵字匹配

目前系統採用**關鍵字匹配**的方式識別學員領域，而非向量（Vector）匹配。具體流程如下：

1. **數據來源**：從學員 IP 地基的 `occupation`（職業/身份）和 `personaExpertise`（專業權威）欄位提取文字
2. **匹配邏輯**：將提取的文字與預設的領域關鍵字表進行比對
3. **分數計算**：統計每個領域匹配到的關鍵字數量，選擇得分最高的作為主要領域

### 領域關鍵字對照表

| 領域 | 關鍵字範例 |
|------|-----------|
| 身心靈 | 療癒、冥想、靈性、能量、心靈、塔羅、占星、覺醒、修行、瑜珈、正念 |
| 商業創業 | 創業、行銷、業務、銷售、品牌、電商、網路、賺錢、被動收入、副業、斜槓 |
| 職場發展 | 職場、工作、升遷、轉職、履歷、面試、主管、領導、管理 |
| 親子教育 | 親子、教育、育兒、媽媽、爸爸、孩子、教養、學習 |
| 兩性關係 | 愛情、感情、婚姻、約會、分手、復合、曖昧、關係 |
| 健康養生 | 健康、養生、減肥、瘦身、運動、健身、飲食、營養 |
| 投資理財 | 投資、理財、股票、基金、房地產、財務自由、存錢 |
| 設計創意 | 設計、創意、美學、藝術、攝影、插畫、品牌設計 |
| 科技技術 | 程式、工程師、AI、科技、軟體、開發、技術 |

### 信心度計算

信心度 = min(1, 匹配關鍵字數 / 5)

例如：若學員的 IP 地基中包含「療癒」、「冥想」、「能量」三個關鍵字，則信心度為 0.6。

## 三、數據應用場景

### 應用 1：腦力激盪（Brainstorm）

**功能位置**：發文工作室 → 引導模式 → 腦力激盪

**應用方式**：
- 當學員輸入主題後，系統使用 `findSimilarViralExamples()` 函數
- 透過 **Embedding 語意匹配**，從 2,542 個向量中找出最相似的爆款貼文
- 將相似的爆款主題作為參考，提供給 AI 生成更精準的建議

**解決的問題**：
| 優化前 | 優化後 |
|--------|--------|
| AI 建議較為泛化，缺乏數據支撐 | 建議基於真實爆款數據，更具參考價值 |
| 無法知道哪些主題容易爆 | 可以看到相似主題的實際爆款案例 |

---

### 應用 2：切角分析（Analyze Angles）

**功能位置**：發文工作室 → 引導模式 → 分析切角

**應用方式**：
- 系統使用 `findSimilarViralExamples()` 找出相似素材的爆款切角
- 從 50 個 IP 的成功案例中，提取切角模式供 AI 參考

**解決的問題**：
| 優化前 | 優化後 |
|--------|--------|
| 切角建議較為通用 | 切角建議來自真實爆款的切入方式 |
| 學員不知道什麼切角有效 | 可以看到相似素材的成功切角案例 |

---

### 應用 3：開頭生成（Generate Hooks）

**功能位置**：發文工作室 → 引導模式 → 生成開頭

**應用方式**：
- 系統使用 `findSimilarViralExamples()` 找出相似主題的爆款開頭
- 從 1,780 篇爆款貼文中，提取開頭模式供 AI 參考

**解決的問題**：
| 優化前 | 優化後 |
|--------|--------|
| 開頭風格較為單一 | 開頭風格多樣，來自不同 IP 的成功模式 |
| 容易產生同質化的開頭 | 透過多樣性排序，確保開頭差異化 |

---

### 應用 4：全文生成（Generate Draft）

**功能位置**：發文工作室 → 引導模式 → 生成全文

**應用方式**：
- 系統使用 `getSpiritualSuccessFactors()` 取得成功因素
- 系統使用 `getContentTypeRecommendations()` 取得內容類型建議
- 系統使用 `getPresentationRecommendations()` 取得呈現方式建議
- 將 603 個成功因素整合到 AI 的 System Prompt 中

**解決的問題**：
| 優化前 | 優化後 |
|--------|--------|
| AI 依靠通用規則生成內容 | AI 依據 50 個頂級 IP 的成功模式生成內容 |
| 無法知道什麼呈現方式更有效 | 數據顯示「含數字」爆款率 41%、「含問句」爆款率 49% |
| 文章長度無依據 | 數據顯示「中等長度 100-200 字」爆款率 62% |

---

### 應用 5：帳號健康度診斷（P2 新功能）

**功能位置**：Dashboard → 帳號健康度

**應用方式**：
- 系統分析學員的發文類型分佈
- 對比 50 個 IP 的黃金比例（情緒 50% / 互動 30% / 品牌 15% / 變現 5%）
- 給出個人化的內容組合建議

---

### 應用 6：今日推薦選題（P2 新功能）

**功能位置**：Dashboard → 今日推薦選題

**應用方式**：
- 系統識別學員的領域（關鍵字匹配）
- 結合學員的經營階段和近期發文類型
- 從成功因素中推薦適合的選題方向

## 四、成功因素分析

### 四大維度統計

| 維度 | 因素數量 | 平均爆款率 | 說明 |
|------|----------|------------|------|
| **呈現方式（Presentation）** | 163 | 5.79% | 數字、問句、驚嘆號等 |
| **主題選擇（Topic）** | 147 | - | 選題方向 |
| **選題角度（Angle）** | 147 | - | 切入角度 |
| **內容類型（Content Type）** | 146 | 5.81% | 長度、結構等 |

### Top 成功因素

| 排名 | 維度 | 因素名稱 | 爆款數 | 爆款率 |
|------|------|----------|--------|--------|
| 1 | 呈現方式 | 含數字 | 450 | 41.28% |
| 2 | 呈現方式 | 含問句 | 303 | 49.11% |
| 3 | 內容類型 | 短文（≤100字） | 252 | 29.03% |
| 4 | 內容類型 | 中等長度（100-200字） | 196 | 61.83% |

### 關鍵發現

1. **含問句的貼文爆款率最高（49.11%）**：在開頭或內文中使用問句，能有效提高互動率
2. **中等長度（100-200字）爆款率達 61.83%**：不要太短也不要太長，這個長度最容易爆
3. **含數字的貼文爆款率 41.28%**：使用具體數字能增加說服力和可讀性

## 六、Embedding 向量應用

### 統計數據

| 指標 | 數值 |
|------|------|
| 有 Embedding 的帳號數 | 49 |
| 總 Embedding 數量 | 2,542 |
| 平均每帳號 Embedding | 52 |

### 語意匹配流程

1. 學員輸入主題或素材
2. 系統將輸入轉換為 Embedding 向量（256 維）
3. 計算與 2,542 個爆款貼文向量的餘弦相似度
4. 返回最相似的 3-5 篇爆款作為參考

### 應用效果

- **主題匹配**：輸入「冥想」，系統會找出所有與冥想相關的爆款貼文
- **風格匹配**：輸入一段文字，系統會找出風格相似的爆款作為參考
- **跨領域學習**：即使學員是新手，也能從頂級 IP 的成功案例中學習

## 八、未來優化方向

### 領域識別升級

目前使用關鍵字匹配，未來可考慮：
1. **向量匹配**：將學員 IP 地基轉為 Embedding，與 50 個 IP 的向量比對
2. **LLM 分類**：使用 AI 直接分析學員的領域和風格
3. **行為學習**：根據學員的發文歷史自動調整領域識別

### 成功因素擴展

1. **動態更新**：當學員產生爆款時，自動提取成功因素
2. **個人化權重**：根據學員領域調整成功因素的權重
3. **A/B 測試**：追蹤不同成功因素的實際效果
