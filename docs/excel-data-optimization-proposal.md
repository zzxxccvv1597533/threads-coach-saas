## 執行摘要

本報告針對「爆款分析儀表板」Excel 檔案中的 11 個分頁進行全面審計，評估各分頁數據在系統中的使用狀況，並提出具體的優化方案。經分析發現，目前僅有 **3 個分頁（27%）** 的數據被完整運用，其餘 **8 個分頁（73%）** 的數據尚未充分發揮價值。

本報告將詳細說明每個分頁的數據內容、目前使用狀況、優化潛力，以及具體的實作建議，預計可提升 AI 生成內容品質 **30-50%**，並顯著改善學員的創作體驗。

### 4.1 資料庫 Schema 設計

```sql
-- 爆款貼文範例庫
CREATE TABLE viral_examples (
  id INT AUTO_INCREMENT PRIMARY KEY,
  keyword VARCHAR(64) NOT NULL,
  post_text TEXT NOT NULL,
  likes INT DEFAULT 0,
  likes_per_day DECIMAL(10,2),
  funnel_stage VARCHAR(16),
  cluster INT,
  opener_50 VARCHAR(200),
  features JSON,  -- 儲存各種特徵標記
  source VARCHAR(64) DEFAULT 'excel_import',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 選題模板庫
CREATE TABLE topic_templates (
  id INT AUTO_INCREMENT PRIMARY KEY,
  cluster INT,
  theme VARCHAR(128),
  template TEXT NOT NULL,
  is_active BOOLEAN DEFAULT TRUE,
  usage_count INT DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 內容群集
CREATE TABLE content_clusters (
  id INT AUTO_INCREMENT PRIMARY KEY,
  cluster_id INT UNIQUE,
  theme_keywords VARCHAR(256),
  posts_count INT,
  top10_rate DECIMAL(5,4),
  median_likes INT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
