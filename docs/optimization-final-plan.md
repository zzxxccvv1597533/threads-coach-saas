# 幕創行銷 Threads AI 教練 - 最終優化方案

## 核心原則

1. **保留現有架構**：不改變「沒靈感」「有素材」「變現內容」三種模式
2. **保留十大貼文類型**：在現有類型基礎上增加爆款元素建議
3. **IP 地基欄位不變**：所有現有欄位維持原樣
4. **增加引導**：幫助學員理解「為什麼文章沒效果」並提供改進方向

---

## 一、優化項目清單

### 1.1 AI 提示詞系統優化

| 項目 | 說明 | 影響範圍 |
|------|------|----------|
| 整合爆款文案結構 | 將分析的爆款文案模式注入 AI 提示詞 | 所有 AI 生成 |
| 十大類型爆款建議 | 每種貼文類型提供可加入的爆款元素 | 發文工作室 |
| 禁止詞彙過濾 | 避免 AI 生成「讓我們」「一起來」等詞彙 | 所有 AI 生成 |
| Threads 風格強化 | 確保輸出符合 Threads 平台風格 | 所有 AI 生成 |

### 1.2 痛點矩陣優化

| 項目 | 說明 | 影響範圍 |
|------|------|----------|
| 生成格式調整 | 改為生成「受眾會說的一句話」 | 痛點矩陣生成 |
| 加入引號格式 | 輸出格式：「貓咪一直亂尿床，是不是討厭我？」 | 痛點矩陣顯示 |

### 1.3 發文工作室引導強化

| 項目 | 說明 | 影響範圍 |
|------|------|----------|
| 貼文類型說明卡 | 選擇類型時顯示該類型的爆款要素和範例 | 類型選擇器 |
| 生成前提示 | 提示學員可加入的爆款元素 | 生成按鈕前 |
| 生成後診斷 | 顯示「這篇文章的優勢」和「可加強的地方」 | 生成結果後 |
| 開頭風格選擇 | 提供 3 種不同風格的開頭讓學員選擇 | 生成結果後 |

### 1.4 文案健檢優化

| 項目 | 說明 | 影響範圍 |
|------|------|----------|
| 問題診斷說明 | 每個扣分項目說明「為什麼這樣不好」 | 健檢結果 |
| 改進建議 | 提供具體的改進方向和範例 | 健檢結果 |
| AI 幫我修改 | 一鍵讓 AI 根據診斷結果優化文案 | 健檢頁面 |

---

## 二、十大貼文類型 × 爆款元素對照表

| 類型 | 建議加入的爆款元素 |
|------|---------------------|
| **故事型** | 開頭用「昨天/上週」具體時間、有轉折點、結尾帶反思 |
| **金句型** | 開頭直接金句、用對比製造張力、結尾問讀者「你呢？」 |
| **知識型** | 開頭用數字或問題、分點清晰、結尾給行動建議 |
| **整理型** | 標題點出數量、每點獨立成金句、可收藏性高 |
| **系列型** | 開頭預告系列、製造期待感、結尾引導追蹤 |
| **問答型** | 開頭用讀者常問的問題、回答要有觀點、結尾反問讀者 |
| **雜談型** | 開頭用日常場景、語氣輕鬆、結尾引發討論 |
| **對話型** | 用「」呈現對話、有情緒轉折、結尾帶感悟 |
| **民調型** | 開頭拋出爭議性選擇、選項要有趣、引導留言 |
| **綜合型** | 結合故事+金句、有情緒起伏、結尾有 CTA |

## 三、不會改變的項目

以下項目在本次優化中**完全不會改變**：

| 項目 | 說明 |
|------|------|
| IP 地基所有欄位 | 職業、語氣、人設三支柱、英雄旅程、受眾、產品等 |
| 發文工作室三種模式 | 沒靈感、有素材、變現內容 |
| 十大貼文類型 | 故事型、金句型、知識型等十種類型 |
| 草稿庫功能 | 草稿 CRUD、串文、Hook 優化 |
| 每日任務系統 | 任務生成、完成追蹤 |
| 數據戰報 | 發文記錄、週報分析 |
| 管理後台 | 學員管理、邀請碼管理 |
| 認證系統 | 登入、註冊、登出 |

## 六、預期效果

優化完成後，學員將能夠：

1. **選擇貼文類型時**：看到該類型的爆款要素，知道「這種文章要怎麼寫才有效」
2. **生成文章後**：看到診斷結果，知道「這篇文章哪裡好、哪裡可以更好」
3. **選擇開頭風格**：從 3 種不同風格中選擇最適合的開頭
4. **文案健檢時**：不只看到分數，還知道「為什麼扣分」和「怎麼改進」
5. **一鍵優化**：讓 AI 根據診斷結果自動優化文案

這樣學員就能在創作過程中持續學習，逐漸理解「什麼樣的文章有效果」。
