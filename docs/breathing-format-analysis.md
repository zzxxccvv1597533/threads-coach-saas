# 爆款文案呼吸感排版分析

## 從爆款 PDF 中提取的排版規則

### 核心發現：真正的呼吸感排版

觀察爆款文案的排版特點，發現以下規律：

#### 1. 極短句原則
- 每句話通常只有 **5-15 個字**
- 一個概念一行，絕不拖泥帶水
- 例如：
  ```
  你最近的磁場超乾淨
  乾淨到好事會黏上你
  壞事不敢靠近
  ```

#### 2. 強制換行原則
- **每一句話都獨立成行**
- 不是「2-3 行空一行」，而是「每行都是獨立的」
- 段落之間用空行分隔，但段落內每句也要換行

#### 3. 節奏變化
- 長短句交錯，創造閱讀節奏
- 短句製造停頓感
- 例如：
  ```
  錢快花光的時候，別慌。
  往往在那個時候，會突然有一筆「剛剛好」的錢進帳。
  當你覺得快撐不下去、準備放棄時，
  也總會有人在你最需要的時刻伸手拉你一把。
  這不是運氣，
  而是你過去善念與福報的迴響。
  ```

#### 4. 情緒遞進
- 每行都有情緒推進
- 用轉折詞創造期待感：「但是」「然而」「沒想到」
- 用因果詞建立邏輯：「因為」「所以」「於是」

#### 5. 清單型的呼吸感
- 清單項目也要有呼吸感
- 每個項目獨立成行
- 項目之間可以有空行
- 例如：
  ```
  你開始轉運的五個徵兆
  
  1.心態變得輕鬆，不再被小事困住
  2.不合適的緣分開始斷離
  3.睡眠開始變好、精神變穩
  4.直覺變強、感覺做什麼都順
  5.開始出現小確幸、小幸運
  ```

### 爆款範例分析

#### 範例 1：玄學提醒型（3.9w 互動）
```
如果你覺得你的2025年是人生中過的最糟糕的年份之一，別擔心！因為你的2026年會迎來徹底的反轉，成為最美好的一年。

這是因為依照神秘學來看，我們剛結束一個九年的循環，如果你對數字學有了解，就會知道9代表完成、結束，以及為新階段清出空間。

簡單來說，今年就是用來清理那些未來不再適合你的人事物，這也是為什麼今年你會覺得充滿失望、或者做什麼都不順，新的循環將在2026年開始，2026是第一年，真正的全新起點，也是我們一直在等的新開始。
```

**問題**：這篇雖然互動高，但排版較密集，不是最佳呼吸感範例。

#### 範例 2：極致呼吸感（6.2w 互動）
```
你最近的磁場超乾淨

乾淨到好事會黏上你

壞事不敢靠近
```

**特點**：
- 每句獨立成行
- 句子極短（5-10 字）
- 段落之間有空行
- 閱讀起來像在呼吸

#### 範例 3：故事型呼吸感（4.7w 互動）
```
我朋友說，不插手別人的選擇，就是一種修行。

以前她總是忍不住給建議、幫忙分析，甚至暗希望對方，照她說的做。

但後來她學會了：每個人都有自己的路要走，哪怕那條路看起來彎、甚至可能會跌倒，那也是對方的靈魂需要經歷的風景。

她說，真正的尊重，並非你幫他選對路，是你願意放手，相信他會在自己的節奏裡，依然走的很好。

她現在是：我愛你，但我不插手不主導。

因為你靜靜地看著，就是相信：他走的每一步，都是被祝福的安排。
```

**特點**：
- 每個段落 1-2 句話
- 段落之間有空行
- 有情緒遞進和轉折
- 結尾有金句收尾

### 呼吸感排版的核心規則

1. **字數限制**：每句不超過 15 字（理想是 10 字以內）
2. **強制換行**：每句話獨立成行
3. **空行規則**：每 2-3 句空一行，或每個段落空一行
4. **節奏變化**：長短句交錯，不要全是短句或全是長句
5. **情緒推進**：每行都要有情緒推進，不要平鋪直敘

### 與現有系統的差距

目前系統的問題：
- 提示詞寫「每 2-3 行空一行」，但沒有強制每句換行
- 句子長度限制是 20 字，應該改為 15 字
- 沒有強調「長短句交錯」的節奏感
- 沒有禁止「文字牆」（連續多行不空行）

### 優化建議

1. **強化排版規則**：
   - 每句不超過 15 字
   - 每句獨立成行
   - 每 2-3 句必須空一行
   - 禁止連續超過 3 行不空行

2. **加入節奏感**：
   - 長短句交錯
   - 開頭用短句抓注意力
   - 中間用稍長句展開
   - 結尾用短句收尾

3. **情緒遞進**：
   - 每行都要有情緒推進
   - 用轉折詞創造期待感
   - 用因果詞建立邏輯
